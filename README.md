# Projeto de Arquitetura e Organização de Computadores I - Processador

Este repositório contém o desenvolvimento incremental de um processador, onde cada atualização adiciona ou evolui um componente específico da arquitetura.

O objetivo é documentar e versionar todo o processo de construção, desde os blocos básicos até a integração final.

## Objetivo do Projeto

* **Construção do Zero:** Projeto e implementação de um processador partindo do início.
* **Modularidade:** Cada componente é desenvolvido de forma isolada e independente.
* **Histórico Estruturado:** Commits definidos para facilitar o entendimento da evolução da arquitetura.
* **Base de Estudo:** Servir como material de referência para aprendizado e expansões futuras.

## Estrutura do Repositório

Cada pasta ou arquivo representa um componente específico do processador (ex: ULA, Registradores, Unidade de Controle). A organização segue a ordem de evolução do hardware.
